Tarea 1


Mire Más

Crear base de datos con llaves foraneas en Mysql
linea de comandos.

Para esto se utilizaran las herramientas:

PuTTy en su ver. 0.67. -Descárgalo-

Servidor Ubuntu LAMP 16.04

La base de datos que se construirá será la siguiente:

Abrir PuTTy y loguearse con root como usuario y su contraseña.

Se mostrara esta ventana si todo anda bien.

Para poder entrar a mysql en su servidor escriba:
mysql –u root –p
en la línea de comando como se muestra,
al presionar enter esta pedirá la contraseña de mysql.

Se mostrara esta ventana si todo anda bien.
Ahora estamos en mysql.

Lo primero será crear la base de datos, escriba:
CREATE DATABASE Cosina;
y presione enter.

Pasaremos a seleccionar la base de datos con la que trabajaremos, escribimos:
USE Cosina;
y presionamos enter.

Ahora crearemos las tablas, para ello un consejo,
para realizar el paste desde el clipboard presione las teclas shift + insert.
Pasamos a copiar los datos de la tabla TIngredReceta.
Y presione enter.

De igual manera los datos de la tabla TReceta.

Y la tabla TIngredientes.

Para revisar que la tabla exista, escriba:
show databases;

Mostrara las bases de datos existentes.

Para revisar las tablas que se encuentran en la base de datos escribimos:
use Cosina;
show tables;

Y nos mostrara las tablas de esta base de datos.

Para ver los campos de cada tabla escribimos:
describe TIngredReceta;
describe TIngredientes;
describe TReceta;

Con esto terminamos para salir de Mysql escribimos:
quit
y presionamos enter.

Y para salir de PuTTy escribimos:
exit
y presionamos enter.

Este se cerrara automaticamente.
