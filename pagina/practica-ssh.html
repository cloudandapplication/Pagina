Mire Más

Conexión por ssh.

Para esto se utilizaran las herramientas:

PuTTy en su ver. 0.67. -Descárgalo-

MySQL Workbench.-Descárgalo-

Servidor Ubuntu LAMP 16.04

Para poder conectarse por SSH necesitaremos primero crear un usuario con todos los permisos. Empezaremos por iniciar sesión desde nuestro servidor, nosotros usaremos Putty para esto, ya al estar dentro de nuestro servidor, procederemos a entrar a mysql mediante la siguiente línea de comandos:
mysql -u root -p

A continuación, necesitaremos ingresar la contraseña de mysql.

Después de esto, estaremos dentro de mysql.

Procederemos a crear nuestro usuario, para esto necesitaremos teclear la siguiente línea de comandos:
CREATE USER 'nombre_usuario'@'%' IDENTIFIED BY 'tu_contrasena';

Acto seguido, le proporcionaremos el acceso a nuestro usuario creado con la siguiente línea de código:
GRANT ALL PRIVILEGES ON * . * TO 'nombre_usuario'@'%';

Después de haberle otorgado los permisos al usuario, refrescaremos todos los privilegios con el siguiente comando:
FLUSH PRIVILEGES;

Con esto terminaremos con el servidor, ahora pasaremos a utilizar MySQL Workbench para realizar la conexión, al iniciar daremos clic en el símbolo de “+” como se muestra en la siguiente imagen:

Al realizar esto, se mostrará la ventana para nuevas conexiones, por default el método es TCP/IP lo debemos de cambiar a SSH

Seguiremos llenando los campos con los datos necesarios, en Connection Name escribiremos el nombre que queremos darle a nuestra conexión, en SSH Hostname ingresaremos la dirección Ip de nuestro servidor, en SSH Username escribiremos el usuario root, en SSH Password escribiremos la contraseña de nuestro servidor, en MySQL Hostname dejaremos 127.0.0.1, en MySQL Server Port dejaremos el puerto 3306, en Username escribiremos el usuario que creamos, en Password escribiremos la contraseña que le creamos al usuario, al haber llenado todos estos campos, procederemos a dar clic en Test Connection como se muestra en la siguiente imagen:

Por último, se mostrará un mensaje confirmando la conexión a MySQL.
