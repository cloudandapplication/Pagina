PuTTy en su ver. 0.67. -Descárgalo-

Git.-Descárgalo-

Servidor Ubuntu LAMP 16.04

Continuaremos con la segunda parte sobre utilizar Git en nuestro servidor. Deberemos posicionarnos en una carpeta en específico en el servidor con el siguiente comando:
cd /var/www/html/

Acto seguido, procederemos a clonar nuestro repositorio GitHub dentro de esta dirección con la siguiente línea de código:
git clone Direccion_Repostorio

Después de haber clonado nuestro repositorio, procederemos a posicionarnos en la carpeta de nuestro repositorio clonado en el server con la siguiente línea de código:
cd /var/www/html/repositorio

Al habernos posicionado en la carpeta, modificaremos el document root de Apache, esto para que funcione el repositorio como página principal, deberemos posicionarnos en una dirección específica para esto:
cd /etc/apache2/sites-avalaibles/

Debemos seleccionar el archivo a modificar mediante la siguiente línea de código:
sudo nano 000-default.conf

Pasaremos a descargar los cambios en nuestro servidor, primero nos posicionaremos en la dirección donde tenemos nuestro repositorio en el servidor.

Ya posicionados en la carpeta, teclearemos la siguiente línea de comando:
